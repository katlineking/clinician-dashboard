Using the ReSpire FD Band Review Interface

Quick instructions for connecting the FD Band, importing device data, and reviewing cough events.

Before You Begin

  • Use Google Chrome or Microsoft Edge.
  • Close Arduino Serial Monitor, Serial Plotter, and any other program using the FD Band COM port.
  • Connect the FD Band to the computer with the USB cable.
  • Select the correct patient record before starting the review.

Connect the FD Band

  1. Click Connect FD Band.
  2. Choose the FD Band serial device when the browser asks for permission.
  3. Wait for the button to change to FD Band Connected.
  4. The website will sync the device clock automatically after connection.

Import and Review Data

  1. Click Import From FD Band.
  2. Wait for the confirmation message that device data was imported successfully.
  3. Click Open Clinical Review.
  4. Use the day selector, timeline controls, and tabs to review cough events and context data.

Helpful Device Controls

  • Pause FD Band temporarily stops sensing and logging.
  • Resume FD Band restarts sensing and logging.
  • clear stored FD Band log erases the stored device log and recreates an empty CSV.
  • download raw FD Band CSV saves the imported raw device data for troubleshooting.

Important Notes

  • For accurate timestamps, connect the FD Band to the website before collecting a monitoring session.
  • If no coughs appear, import device data again and make sure the review is opened afterward.
  • If needed, clear the stored FD Band log before a new test session to avoid mixing old and new entries.
